# AI Forge Path v2

**AI Forge Path v2** — PWA-трекер обучения профессии AI Engineer. Версия 2 перестроена вокруг четырёх вопросов:

1. Что я уже выполнил?
2. Что находится в работе прямо сейчас?
3. Что нужно сделать следующим действием?
4. Насколько подтверждён каждый навык?

## Механики обучения v2.1

- серия засчитывается за одно осмысленное действие, а дневная цель считается отдельно;
- до двух автоматических заморозок серии;
- ежедневные квесты и сундук наград;
- XP начисляется за продвижение, практику, проверку знаний, повторение и проекты;
- персональная недельная лига без искусственного социального рейтинга;
- банк ошибок и адаптивный Practice Hub;
- интервальные повторения через 1, 3, 7, 14, 30 и 60 дней;
- двухминутная сессия для занятых дней;
- достижения и награды без блокировки обучения за ошибки.

## Главное в версии 2

- пять понятных статусов темы: **не начато → изучаю → практика → проверка → освоено**;
- подробная история действий с датой и временем;
- оценка общего уровня AI Engineering по шкале 0–100;
- отдельная оценка 11 направлений: Python, AI, LLM, RAG, Agents, Backend, Deployment и другие;
- стартовая самооценка, которая задаёт только начальную базу;
- более высокий вес реальной практики, проверок, повторения и проектов;
- главный экран с одной текущей темой и одним очевидным следующим действием;
- планы на сегодня, неделю и полный roadmap;
- 6 проектов-доказательств для портфолио;
- автоматическое сохранение и локальные резервные копии;
- экспорт и импорт полного прогресса в JSON;
- миграция данных из первой версии приложения;
- офлайн-режим и установка на телефон.

## Где хранится прогресс

Основные данные хранятся в **IndexedDB** браузера. Дополнительная резервная копия состояния сохраняется в `localStorage` под ключом:

```text
ai-forge-path-state-v2
```

Закрытие приложения, перезагрузка страницы или публикация новой версии **не сбрасывают прогресс**.

Прогресс может исчезнуть при очистке данных браузера, использовании приватного режима или переходе на другое устройство. Для этого в настройках предусмотрены:

- ежедневный локальный снимок;
- ручной локальный снимок;
- экспорт JSON-файла;
- восстановление из JSON.

## Как считается уровень

Каждая тема получает до 100 баллов:

- статус изучения — до 35;
- практика — до 25;
- проверка знаний — до 20;
- повторение — до 10;
- проекты-доказательства — до 10.

Стартовая самооценка задаёт лишь предварительную базу до 24 баллов. Реальные действия постепенно заменяют её подтверждёнными результатами.

Уровни:

| Баллы | Уровень |
|---:|---|
| 0–19 | Новичок |
| 20–39 | AI Explorer |
| 40–59 | AI Builder |
| 60–74 | Junior AI Engineer |
| 75–89 | Applied AI Engineer |
| 90–100 | Production-ready AI Engineer |

Эта оценка предназначена для управления обучением и не является официальной сертификацией.

## Локальный запуск

```bash
python -m http.server 8080
```

```text
http://localhost:8080
```
